## Tools that will help you conduct code reviews

Reviewers can intergrate this tools with Github to get automated code review for test coverage, complexity, duplication, security, style, and more, and merge with confidence. 

- [Codeacy](https://www.codacy.com/ "Codeacy")
- [Hound](https://houndci.com/ "Hound")
- [CodeClimate](https://codeclimate.com/ "CodeClimate")
- [Phabricator]("https://www.phacility.com/" "Phabricator")
- [SmartBear]("https://smartbear.com/product/collaborator/overview/" "SmartBear")
